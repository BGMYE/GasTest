# RuoYi-vue3-flowable

基于 Spring Boot 2 的快速开发框架，集成 Vue 3 和 Flowable 工作流引擎，前端界面经过美化优化。

## 项目特色

- ⚡ 基于 `springboot2` 构建，性能优异
- 🖼 集成 `vue3` 前端框架，用户体验佳
- 🔧 内置 `flowable` 工作流引擎，支持流程管理
- 💄 前端美化，现代化界面设计

## 技术栈

- 后端：Spring Boot 2.x、Flowable 6.x
- 前端：Vue 3.x、Element Plus
- 数据库：MySQL、MyBatis Plus
